# ¿Qué es el Machine Learning?
## Definición
El Machine Learning es el campo que permite a los sistemas aprender patrones a partir de datos sin ser explícitamente programados.

## Ciclo de ML
1. Datos
2. Modelo
3. Entrenamiento
4. Predicción
5. Evaluación

---

# Categorías del Aprendizaje Supervisado
## Algoritmos Clásicos
- Regresión lineal y polinomial
- Clasificación
- k-Nearest Neighbors (kNN)
- Árboles de decisión
- Random Forests
- SVM (mención conceptual)

## Deep Learning (Redes Neuronales)
- Inspiradas en el cerebro
- Aprenden representaciones profundas
- Requieren muchos datos y poder de cómputo

---

# Tipos de Problemas
## Regresión
Predicción de valores numéricos.

## Clasificación
Asignación de una categoría a una entrada.

## Clustering (no supervisado)
Agrupación por similitud sin etiquetas.
